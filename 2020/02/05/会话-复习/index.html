дјҡиҜқ

жҰӮиҝ°

дёҖж¬ЎдјҡиҜқдёӯеҢ…еҗ«еӨҡж¬ЎиҜ·жұӮе’Ңе“Қеә”

  • дёҖж¬ЎдјҡиҜқпјҡжөҸи§ҲеҷЁз¬¬дёҖж¬Ўз»ҷжңҚеҠЎеҷЁиө„жәҗеҸ‘йҖҒиҜ·жұӮпјҢдјҡиҜқе»әз«ӢпјҢзӣҙеҲ°жңүдёҖж–№ж–ӯејҖдёәжӯў
  • еҠҹиғҪпјҡеңЁдёҖж¬ЎдјҡиҜқиҢғеӣҙеҶ…зҡ„еӨҡж¬ЎиҜ·жұӮй—ҙпјҢе…ұдә«ж•°жҚ®
  • ж–№ејҸпјҡ
    1. е®ўжҲ·з«ҜдјҡиҜқжҠҖжңҜпјҡCookie
    2. жңҚеҠЎеҷЁдјҡиҜқжҠҖжңҜпјҡSession

Cookie

  • жҰӮеҝөпјҡе®ўжҲ·з«ҜдјҡиҜқжҠҖжңҜ-е°Ҷж•°жҚ®дҝқеӯҳеңЁе®ўжҲ·з«ҜжөҸи§ҲеҷЁ

  • еҝ«йҖҹе…Ҙй—Ёпјҡ

    • дҪҝз”ЁжӯҘйӘӨпјҡ
      1. еҲӣе»әCookieеҜ№иұЎпјҢз»‘е®ҡж•°жҚ®пјҡnew Cookie(String name,String value)
      2. еҸ‘йҖҒCookieеҜ№иұЎпјҡresp.addCookie(Cookie cookie)
      3. иҺ·еҸ–CookieпјҢжӢҝеҲ°ж•°жҚ®пјҡCookie[] req.getCookies()

  • е®һзҺ°еҺҹзҗҶпјҡ

    • еҹәдәҺе“Қеә”еӨҙset-cookieе’ҢиҜ·жұӮеӨҙcookieе®һзҺ°

  • cookieзҡ„з»ҶиҠӮ

    1. дёҖж¬ЎеҸҜеҗҰеҸ‘йҖҒеӨҡдёӘcookieпјҹ

      • еҸҜд»ҘпјҢйҖҡиҝҮеҲӣе»әеӨҡдёӘCookieеҜ№иұЎпјҢдҪҝз”Ёrespи°ғз”ЁеӨҡж¬ЎaddCookieж–№жі• еҸ‘йҖҒcookieеҜ№иұЎеҚіеҸҜ

    2. cookieеңЁжөҸи§ҲеҷЁдёӯдҝқеӯҳеӨҡй•ҝж—¶й—ҙпјҹ

      • й»ҳи®Өжғ…еҶөдёӢпјҢеҪ“жөҸи§ҲеҷЁе…ій—ӯеҗҺпјҢcookieж•°жҚ®иў«й”ҖжҜҒ
      • жҢҒд№…еҢ–еӯҳеӮЁпјҡsetMaxAge(int seconds)
        • seconds>0пјҡе°ҶCookieж•°жҚ®еҶҷеҲ°зЎ¬зӣҳзҡ„ж–Үд»¶дёӯпјҢжҢҒд№…еҢ–еӯҳеӮЁгҖӮsecondsд»ЈиЎЁCookieеӯҳжҙ»ж—¶й—ҙ
        • second<0пјҡй»ҳи®ӨеҖј
        • second=0пјҡеҲ йҷӨcookieдҝЎжҒҜ

    3. cookieиғҪеҗҰеӯҳдёӯж–Ү

      • еңЁtomcat 8 д№ӢеүҚпјҢcookieдёҚиғҪзӣҙжҺҘеӯҳеӮЁдёӯж–Үж•°жҚ®пјҢйңҖиҰҒе°Ҷдёӯж–Үж•°жҚ®иҪ¬з ҒвҖ”-дёҖиҲ¬йҮҮз”ЁURLзј–з Ғ(%E3)
      • еңЁtomcat 8 д№ӢеҗҺпјҢcookieеҸҜд»ҘеӯҳеӮЁдёӯж–Үж•°жҚ®пјҢдҪҶд»Қз„¶дёҚж”ҜжҢҒзү№ж®Ҡеӯ—з¬ҰпјҢйҮҮз”ЁURLзј–з Ғ

    4. cookieе…ұдә«й—®йўҳпјҹ

      1. еҒҮи®ҫеңЁдёҖдёӘtomcatжңҚеҠЎеҷЁдёӯпјҢйғЁзҪІеӨҡдёӘйЎ№зӣ®пјҢеҲҷиҝҷдәӣйЎ№зӣ®зҡ„cookieиғҪеҗҰе…ұдә«пјҹ
        • й»ҳи®Өжғ…еҶөдёӢcookieдёҚиғҪе…ұдә«
        • setPath(String path)пјҡи®ҫзҪ®cookieзҡ„иҺ·еҸ–иҢғеӣҙпјҢй»ҳи®Өжғ…еҶөдёӢи®ҫзҪ®еҪ“еүҚзҡ„иҷҡжӢҹзӣ®еҪ•
        • еҰӮжһңйңҖиҰҒе…ұдә«пјҢеҲҷеҸҜе°Ҷpathи®ҫзҪ®дёәвҖқ/вҖң
      2. дёҚеҗҢжңҚеҠЎеҷЁй—ҙcookieе…ұдә«й—®йўҳпјҹ(жҜ”еҰӮзҷҫеәҰжҗңзҙўдёҺзҷҫеәҰиҙҙжқҝзӯүд№Ӣй—ҙ)
        • setDomain(String path)пјҡеҰӮжһңи®ҫзҪ®дёҖзә§еҹҹеҗҚзӣёеҗҢпјҢйӮЈд№ҲеӨҡдёӘжңҚеҠЎеҷЁд№Ӣй—ҙcookieеҸҜд»Ҙе…ұдә«

    5. Cookieзҡ„зү№зӮ№е’ҢдҪңз”Ё

      1. cookieеӯҳеӮЁж•°жҚ®еңЁе®ўжҲ·з«ҜжөҸи§ҲеҷЁ
      2. жөҸи§ҲеҷЁеҜ№дәҺеҚ•дёӘCookieзҡ„еӨ§е°ҸжңүйҷҗеҲ¶(4kb)д»ҘеҸҠеҜ№еҗҢдёҖеҹҹеҗҚзҡ„жҖ»cookieж•°йҮҸд№ҹжңүйҷҗеҲ¶(20дёӘ)
      • дҪңз”Ёпјҡ
        1. cookieдёҖиҲ¬з”ЁдәҺеӯҳеӮЁе°‘йҮҸзҡ„дёҚеӨӘж•Ҹж„ҹзҡ„ж•°жҚ®
        2. еңЁдёҚзҷ»еҪ•зҡ„жғ…еҶөдёӢпјҢе®ҢжҲҗжңҚеҠЎеҷЁеҜ№е®ўжҲ·з«Ҝзҡ„иҜҶеҲ«

    6. жЎҲдҫӢ-и®°дҪҸдёҠж¬Ўи®ҝй—®ж—¶й—ҙ

Session

  • жҰӮеҝөпјҡжңҚеҠЎеҷЁз«ҜдјҡиҜқжҠҖжңҜпјҢеңЁдёҖж¬ЎдјҡиҜқзҡ„еӨҡж¬ЎиҜ·жұӮй—ҙе…ұдә«ж•°жҚ®пјҢе°Ҷж•°жҚ®дҝқеӯҳеңЁжңҚеҠЎеҷЁз«Ҝзҡ„еҜ№иұЎдёӯгҖӮHttpSession

  • еҝ«йҖҹе…Ҙй—Ёпјҡ

    1. иҺ·еҸ–sessionеҜ№иұЎ
      • HttpSession session = req.getSession()
    2. дҪҝз”ЁHttpSessionеҜ№иұЎпјҡ
      • Object getAttribute(String name)
      • void setAttribute(String name,Object value)
      • void removeAttribute(String name)

  • еҺҹзҗҶпјҡ

    • sessionзҡ„е®һзҺ°жҳҜдҫқиө–дәҺcookieзҡ„

  • з»ҶиҠӮ

    1. еҪ“е®ўжҲ·з«Ҝе…ій—ӯеҗҺпјҢжңҚеҠЎеҷЁдёҚе…ій—ӯпјҢдёӨж¬ЎиҺ·еҸ–sessionжҳҜеҗҰдёәеҗҢдёҖдёӘ?

      • й»ҳи®Өжғ…еҶөдёӢдёҚжҳҜ

      • еҰӮжһңйңҖиҰҒзӣёеҗҢпјҢеҲҷеҸҜд»ҘеҲӣе»әCookieпјҢй”®дёәJSESSIONIDпјҢи®ҫзҪ®жңҖеӨ§еӯҳжҙ»ж—¶й—ҙпјҢи®©cookieжҢҒд№…еҢ–дҝқеӯҳ

  1. е®ўжҲ·з«ҜдёҚе…ій—ӯпјҢжңҚеҠЎеҷЁе…ій—ӯеҗҺпјҢдёӨж¬ЎиҺ·еҸ–зҡ„sessionжҳҜеҗҰдёәеҗҢдёҖдёӘпјҹ

    • дёҚжҳҜеҗҢдёҖдёӘпјҢдҪҶжҳҜиҰҒзЎ®дҝқж•°жҚ®дёҚдёўеӨұ
      • sessionзҡ„й’қеҢ–пјҡ
        • еңЁжңҚеҠЎеҷЁжӯЈеёёе…ій—ӯд№ӢеүҚпјҢе°ҶsessionеҜ№иұЎзі»еҲ—еҢ–еҲ°зЎ¬зӣҳдёҠ
      • sessionзҡ„жҙ»еҢ–пјҡ
        • еңЁжңҚеҠЎеҷЁеҗҜеҠЁеҗҺпјҢе°Ҷsessionж–Үд»¶иҪ¬еҢ–дёәеҶ…еӯҳдёӯзҡ„sessionеҜ№иұЎеҚіеҸҜ
    • tomcatдјҡиҮӘеҠЁе®ҢжҲҗй’қеҢ–е’Ңжҙ»еҢ–пјҢдҪҶideaдёӯж— жі•е®ҢжҲҗ

  2. sessionзҡ„еӨұж•Ҳж—¶й—ҙпјҹ

    1. жңҚеҠЎеҷЁе…ій—ӯ

    2. sessionеҜ№иұЎи°ғз”Ёinvalidate()

    3. sessionй»ҳи®ӨеӨұж•Ҳж—¶й—ҙжҳҜ30еҲҶй’ҹ

  • sessionзҡ„зү№зӮ№

    1. sessionз”ЁдәҺеӯҳеӮЁдёҖж¬ЎдјҡиҜқзҡ„еӨҡж¬ЎиҜ·жұӮзҡ„ж•°жҚ®пјҢеӯҳеңЁжңҚеҠЎеҷЁз«Ҝ
    2. sessionеҸҜд»ҘеӯҳеӮЁд»»ж„Ҹзұ»еһӢгҖҒд»»ж„ҸеӨ§е°Ҹзҡ„ж•°жҚ®
