Filterе’ҢListener

Filter:иҝҮж»ӨеҷЁ

  • еҪ“и®ҝй—®жңҚеҠЎеҷЁдёӯзҡ„иө„жәҗж—¶пјҢиҝҮж»ӨеҷЁеҸҜд»Ҙе°ҶиҜ·жұӮжӢҰжҲӘдёӢжқҘпјҢе®ҢжҲҗдёҖдәӣзү№ж®Ҡзҡ„еҠҹиғҪ

  • иҝҮж»ӨеҷЁзҡ„дҪңз”Ёпјҡ

    • дёҖиҲ¬з”ЁдәҺе®ҢжҲҗйҖҡз”Ёзҡ„ж“ҚдҪңгҖӮеҰӮпјҡзҷ»еҪ•йӘҢиҜҒгҖҒз»ҹдёҖзј–з ҒеӨ„зҗҶгҖҒж•Ҹж„ҹеӯ—з¬ҰиҝҮж»ӨвҖҰ

  • еҝ«йҖҹе…Ҙй—Ё

    1. е®ҡд№үдёҖдёӘзұ»пјҢе®һзҺ°жҺҘеҸЈFilter

  1. еӨҚеҶҷж–№жі•

    1. й…ҚзҪ®жӢҰжҲӘи·Ҝеҫ„

      • web.xml
      • жіЁи§Ј

  • иҝҮж»ӨеҷЁз»ҶиҠӮпјҡ

    1. web.xmlй…ҚзҪ®

  1. иҝҮж»ӨеҷЁжү§иЎҢжөҒзЁӢ

    1. жү§иЎҢиҝҮж»ӨеҷЁ
    2. жү§иЎҢж”ҫиЎҢеҗҺзҡ„иө„жәҗ
    3. еӣһжқҘжү§иЎҢиҝҮж»ӨеҷЁж”ҫиЎҢд»Јз ҒдёӢиҫ№зҡ„д»Јз Ғ

  2. иҝҮж»ӨеҷЁз”ҹе‘Ҫе‘Ёжңҹж–№жі•

    1. initпјҡеңЁжңҚеҠЎеҷЁеҗҜеҠЁеҗҺпјҢдјҡеҲӣе»әFilterеҜ№иұЎпјҢз„¶еҗҺи°ғз”Ёinitж–№жі•гҖӮеҸӘжү§иЎҢдёҖж¬ЎгҖӮз”ЁдәҺеҠ иҪҪиө„жәҗ
    2. doFilterпјҡжҜҸж¬ЎиҜ·жұӮиў«жӢҰжҲӘиө„жәҗж—¶пјҢдјҡиў«жү§иЎҢпјҢжү§иЎҢеӨҡж¬Ў
    3. destroyпјҡеңЁжңҚеҠЎеҷЁе…ій—ӯеҗҺпјҢFilterеҜ№иұЎиў«й”ҖжҜҒпјҢеҰӮжһңжңҚеҠЎеҷЁжҳҜжӯЈеёёе…ій—ӯпјҢеҲҷдјҡжү§иЎҢdestroyж–№жі•гҖӮеҸӘжү§иЎҢдёҖж¬ЎпјҢз”ЁдәҺйҮҠж”ҫиө„жәҗ

  3. иҝҮж»ӨеҷЁй…ҚзҪ®иҜҰи§Ј

    1. жӢҰжҲӘи·Ҝеҫ„й…ҚзҪ®

      • е…·дҪ“иө„жәҗи·Ҝеҫ„пјҡ/index.jsp еҸӘжңүи®ҝй—®index.jspиө„жәҗж—¶пјҢиҝҮж»ӨеҷЁиў«жү§иЎҢ
      • жӢҰжҲӘзӣ®еҪ•пјҡ/user/* и®ҝй—®/userдёӢжүҖжңүиө„жәҗж—¶пјҢиҝҮж»ӨеҷЁйғҪдјҡиў«жү§иЎҢ
      • еҗҺзјҖеҗҚжӢҰжҲӘпјҡ*.jsp и®ҝй—®жүҖжңүеҗҺзјҖеҗҚдёәjspиө„жәҗж—¶пјҢиҝҮж»ӨеҷЁйғҪдјҡиў«жү§иЎҢ
      • жӢҰжҲӘжүҖжңүиө„жәҗпјҡ/*

    2. жӢҰжҲӘж–№ејҸй…ҚзҪ®пјҡиө„жәҗиў«и®ҝй—®зҡ„ж–№ејҸ

      • жіЁи§Јй…ҚзҪ®

        • и®ҫзҪ®dispatcherTypes
          1. REQUESTпјҡй»ҳи®ӨеҖјгҖӮжөҸи§ҲеҷЁзӣҙжҺҘиҜ·жұӮиө„жәҗ
          2. FORWARDпјҡиҪ¬еҸ‘и®ҝй—®иө„жәҗ
          3. INCLUDEпјҡеҢ…еҗ«и®ҝй—®иө„жәҗ
          4. ERRORпјҡй”ҷиҜҜи·іиҪ¬иө„жәҗ
          5. ASYNCпјҡејӮжӯҘи®ҝй—®иө„жәҗ

      • web.xml

        <dispatcher></dispatcher>

  4. иҝҮж»ӨеҷЁй“ҫ

    • жү§иЎҢйЎәеәҸпјҡеҰӮжһңжңүдёӨдёӘиҝҮж»ӨеҷЁпјҡиҝҮж»ӨеҷЁ1е’ҢиҝҮж»ӨеҷЁ2
      1. иҝҮж»ӨеҷЁ1жү§иЎҢ
      2. иҝҮж»ӨеҷЁ2жү§иЎҢ
      3. иө„жәҗжү§иЎҢ
      4. иҝҮж»ӨеҷЁ2жү§иЎҢ
      5. иҝҮж»ӨеҷЁ1жү§иЎҢ
    • е…ҲеҗҺйЎәеәҸй—®йўҳпјҡ
      • жіЁи§Јй…ҚзҪ®пјҡжҢүз…§зұ»еҗҚзҡ„еӯ—з¬ҰдёІжҜ”иҫғи§„еҲҷжҜ”иҫғпјҢеҖје°Ҹзҡ„е…Ҳжү§иЎҢ(еҰӮAFilterе’ҢBFilterзӣёжҜ”пјҢеҲҷAFilterе…Ҳжү§иЎҢ)
      • web.xmlй…ҚзҪ®пјҡ< filter-mapping >и°Ғе®ҡд№үеңЁдёҠйқўи°Ғе…Ҳжү§иЎҢ

  • жЎҲдҫӢ-иҝҮж»Өж•Ҹж„ҹиҜҚ

    • дҪҝз”ЁеҠЁжҖҒд»ЈзҗҶжЁЎејҸпјҢйҖҡиҝҮеўһејәrequestеҜ№иұЎиҺ·еҸ–еҸӮж•°зӣёе…іж–№жі•ж•Ҹж„ҹиҜҚиҝҮж»Ө

Listenerпјҡзӣ‘еҗ¬еҷЁ

  • жҰӮеҝөпјҡwebзҡ„дёүеӨ§з»„д»¶д№ӢдёҖ

    • дәӢд»¶зӣ‘еҗ¬жңәеҲ¶
      • дәӢд»¶пјҡдёҖд»¶дәӢжғ…
      • дәӢд»¶жәҗпјҡдәӢд»¶еҸ‘з”ҹзҡ„ең°ж–№
      • зӣ‘еҗ¬еҷЁпјҡдёҖдёӘеҜ№иұЎ
      • жіЁеҶҢзӣ‘еҗ¬пјҡе°ҶдәӢд»¶гҖҒдәӢд»¶жәҗгҖҒзӣ‘еҗ¬еҷЁз»‘е®ҡеңЁдёҖиө·гҖӮеҪ“дәӢд»¶жәҗдёҠеҸ‘з”ҹжҹҗдёӘдәӢд»¶еҗҺпјҢжү§иЎҢзӣ‘еҗ¬еҷЁд»Јз ҒгҖӮ

  • ServletContextListenerпјҡзӣ‘еҗ¬servletContextеҜ№иұЎзҡ„еҲӣе»әе’Ңй”ҖжҜҒ

    • void contextDestroyed(ServletContextEvent sce)пјҡServletContextеҜ№иұЎиў«й”ҖжҜҒд№ӢеүҚдјҡи°ғз”ЁиҜҘж–№жі•
    • void contextInitialized(ServletContextEvent sce)пјҡServletContextеҜ№иұЎеҲӣе»әеҗҺдјҡи°ғз”ЁиҜҘж–№жі•

  • жӯҘйӘӨпјҡ
